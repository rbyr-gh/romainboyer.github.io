Contexte & ObjectifsContext & Goals

Dans de nombreuses applications (sécurité, contrôle d’accès, automatisation domestique ou industrielle), certaines opérations nécessitent une intervention humaine répétitive ou délicate, ce qui peut générer des erreurs ou des risques. Pour améliorer la précision, réduire l’effort et garantir la sécurité, ce projet présente un loquet électronique entièrement conçu et réalisé individuellement, contrôlé via un clavier matriciel. Ce système permet de verrouiller et déverrouiller un mécanisme de manière fiable et rapide, tout en offrant une interface simple et intuitive pour l’utilisateur.

In many applications (security, access control, home or industrial automation), certain operations require repetitive or delicate human intervention, which can lead to errors or risks. To improve accuracy, reduce effort, and ensure safety, this project presents a fully designed and individually developed electronic latch, controlled via a matrix keypad. This system enables reliable and fast locking and unlocking of a mechanism, while providing a simple and intuitive interface for the user.

Architecture techniqueTechnical Architecture

Le loquet électronique est conçu pour offrir un contrôle sécurisé et intuitif. Une interface affiche les informations importantes, permettant à l’utilisateur de savoir si le code saisi est correct. Des LED indiquent visuellement la validité du code : elles s’allument ou s’éteignent en fonction de la réussite ou de l’échec. Lorsqu’un code correct est saisi, la gâche électrique s’ouvre pour permettre l’accès, puis se referme automatiquement après usage. Le système inclut également une sécurité : il se bloque après trois tentatives incorrectes, empêchant toute manipulation non autorisée. Enfin, il est possible de changer le code, offrant une flexibilité et un contrôle supplémentaire à l’utilisateur.

The electronic latch is designed to provide secure and intuitive control. An interface displays key information, allowing the user to verify whether the entered code is correct. LEDs provide visual feedback by turning on or off depending on success or failure. When the correct code is entered, the electric strike unlocks to grant access, then automatically locks again after use. The system also includes a security feature: it locks after three incorrect attempts, preventing unauthorized access. Finally, the code can be changed, offering additional flexibility and control to the user.
